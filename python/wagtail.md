# Getting Wagtail Working

[Django Girls Tutorial](http://tutorial.djangogirls.org/en/)
Great tutorial!!

Start virtual environment:
```bash
source myvenv/bin/activate
```

Stop:
```bash
deactivate
```

Running server locally:
```bash
python manage.py runserver
```
